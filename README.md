# marktile

**Open any Markdown note in tugtile's editor, as a real [Obsidian](https://obsidian.md) pane.**

marktile (mark + tile) is the standalone editor from the
[tugtile](https://github.com/CVERInc/tugtile) family: headings grow while the literal
`## ` markers stay in place, smart-Enter list continuation, **Tab** indentation with a
tab-vs-space ruler, two modes (styled / plain monospace), and a drag-reorderable
**table of contents**. CJK-safe, no rendering layer — just a faster way to edit `.md`.

It's a switchable pane: a header button hops back to Obsidian's native editor, and — if
tugtile is installed — over to the kanban board. Same file, no lock-in.

## Install

- **Manual:** from the [latest release](../../releases/latest), download `main.js`,
  `manifest.json`, and `styles.css` into your vault's `.obsidian/plugins/marktile/` folder,
  then enable **marktile** in Settings → Community plugins.
- **[BRAT](https://github.com/TfTHacker/obsidian42-brat):** add the beta plugin `CVERInc/marktile`.
