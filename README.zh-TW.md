# marktile

**用 tugtile 的編輯器打開任何一份 Markdown 筆記，成為 [Obsidian](https://obsidian.md) 的真分頁。**

marktile（mark + tile）是 [tugtile](https://github.com/CVERInc/tugtile) 家族的獨立編輯器：
`## ` 標記留在原地、標題卻會放大，智慧 Enter 接續清單，**Tab** 縮排（附分辨 tab 與空白的標尺），
兩種模式（調味／原味等寬），以及可拖拉重排的**目錄**。CJK 安全、沒有渲染層——就是更快地編輯 `.md`。

它是一個可切換的分頁：header 上的鈕能跳回 Obsidian 原生編輯器，若有裝 tugtile 還能切到看板。
同一份檔案，沒有鎖定。

## 安裝

- **手動：** 從[最新 Release](../../releases/latest) 下載 `main.js`、`manifest.json`、
  `styles.css` 放進你 vault 的 `.obsidian/plugins/marktile/` 資料夾，再到 設定 → 社群外掛
  啟用 **marktile**。
- **[BRAT](https://github.com/TfTHacker/obsidian42-brat)：** 加入測試版外掛 `CVERInc/marktile`。
